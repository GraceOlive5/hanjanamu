# 🌳 한자나무 (HanjaNamu)

> 한 글자씩 쌓아 올리는 어휘력의 숲

한자 학습을 통한 어휘력 향상 퀴즈 앱

## ✨ 특징

- 📚 8급~1급 체계적 학습 (3,500자)
- 💎 별자리 등급 시스템 (Bronze → Diamond)
- 🎯 실전 활용 단어 제공
- 📊 학습 통계 및 복습 시스템
- 🎨 세련된 UI/UX

## 🚀 기술 스택

- HTML5 / CSS3 / JavaScript
- LocalStorage (데이터 저장)
- JSON (한자 데이터)
- GitHub Pages / Firebase Hosting

## 📱 설치
```bash
git clone https://github.com/grace/hanja-namu.git
cd hanja-namu
# index.html 열기
```
